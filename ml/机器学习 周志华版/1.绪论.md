## 1.2基本术语

要进行机器学习，先要有数据，假定收集了一批关于西瓜的数据，例如：
（色泽 = 青绿； 根蒂 = 蜷缩； 敲声 = 清脆；）、
（色泽 = 乌黑； 根蒂 = 稍蜷； 敲声 = 沉闷；）、
（色泽 = 浅白； 根蒂 = 硬挺； 敲声 = 清脆；）、
其中，每一对括号内就是一条记录，“ = ” 意味 “取值为”。

以上的记录的集合称为**数据集**（date set），其中每条记录是关于一个事件或对象的描述，称为一个**示例**（instance）或**样本**（sample）。

反应时间或对象在某方面的表现或性质的事项，例如“色泽” “根蒂” “敲声”，称为**属性**（attribute）或**特征**（feature）；属性上的取值，例如“青绿” “乌黑”，称为**属性值**。

属性张成一个用于描述西瓜的三维空间，每个西瓜都可以在这个空间中找到自己的坐标位置。由于空间中的每一个点对应一个坐标向量，我们也就把一个示例称为一个**特征向量**（feature vector）.


一般的，令$D = {x_1, x_2, ..., x_m}$表示包含m个示例的数据集，每个示例由d个属性描述，则每个示例$x_i = (x_i1;x_i2;x_i3;...;x_id)$，是d维样本空间 中的一个向量， 其中$x_ij$是在第j个属性上的取值，d称为样本$x_i$的**维数**（dimensionality）。

从数据中学得模型的过程称为**学习**（learning）或**训练**（training），这个过程通过执行某个学习算法来完成。训练过程中使用的数据称为**训练数据**（training data），其中每个样本称为一个**训练样本**（training sample），训练样本组成的集合称为**训练集**（training set）。
